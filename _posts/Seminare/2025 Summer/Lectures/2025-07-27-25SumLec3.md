{% abs %}
A wide range of cotorsion pairs are constructed via covering and enveloping. We analyse the existence of projective covers at the outset, and identify where the obstacle arises. Subsequently, we introduce criteria for the existence of covers and envelopes.

## All Covers are Projective Covers

We explain the general theory of (pre)covers with projective (pre)covers in a functorial approach.

### Functorial Approach

{% lem %}
The Yoneda embedding $\mathcal{A} ‚Ü™ ùêèùêíùê° (\mathcal{A})$ via $X ‚Ü¶ (-, X)_\mathcal{A}$ is fully faithful and left exact. In particular, the **representable functor $(-, X)_\mathcal{A}$ are projective** in $ùêèùêíùê° (\mathcal{A})$.

{% pf %}
The Yoneda embedding yields a bijection $(X,Y) ‚Üî [(-, X), (-, Y)]_{ùêèùêíùê° (\mathcal{A})}$, which identifies $f$ with a collection of natural transformations $\{(A, f)\}_{A ‚àà \mathcal{A}}$.

- The assignment $? ‚Ü¶ (-, ?)$ preserves all limits, and is thus left exact;
- For any surjection $Œ∏ : F ‚Üí G$ of presheaves, we observe $[(-, ?), Œ∏_-] ‚âÉ Œ∏_?$ is surjective. Hence, representable functors are projective in $ùêèùêíùê° (\mathcal{A})$.

{% endpf %}

{% endlem %}

We now provide a functorial characterisation of (pre)cover. Throughout, $\mathcal{F}$ denotes a replete (= closed under isomorphism) class of objects.

{% def %}
((Pre)cover). Let $X ‚àà \mathcal{A}$ and $F ‚àà \mathcal{F}$. We say $f : F ‚Üí X$ is an $\mathcal{F}$-**precover** if the following equivalent conditions hold:

1. Any morphism $g : G ‚Üí X$ with $G ‚àà \mathcal{F}$ factors through $f$;
2. There exists an epimorphism $f_‚àó : (- , F)_\mathcal{F} ‚Ü† (-, X)_\mathcal{A}$ from the projective object $(-, F)_\mathcal{F} ‚àà ùêèùêíùê° (\mathcal{F})$.

A precover $f : F ‚Üí X$ is termed a **cover** if the following equivalent conditions hold:

1. Any endomorphism $Œ± : F ‚Üí X$ such that $f ‚àò Œ± = f$ implies $Œ±$ is an automorphism;
2. The epimorphism $f_‚àó : (-, F)_\mathcal{F} ‚Ü† (-, X)_\mathcal{A}$ is a projective cover.

Note that for $2$, $\mathrm{End}_\mathcal{F}(F) ‚âÉ \mathrm{End}_{ùêèùêíùê° (\mathcal{F})}((-, F)_\mathcal{F})$ via $Œ± ‚Ü¶ Œ± _‚àó$.

{% enddef %}

{% pro %}
Show that projective precovers are precisely those epimorphisms from projective objects.
{% endpro %}

{% pro %}
Show that if $M$ has a $\mathcal{F}$-cover, then it is a summand of any $\mathcal{F}$-precover.
{% endpro %}

From a categorical perspective, to study covers, it suffices to study projective covers.

The functorial construction yields remarkable results for AR theory, e.g., [this note](20250523Talk#‰∏é-ar-ÁêÜËÆ∫ÁöÑÂÖ≥Á≥ª), or a general theory of Auslander algebra.

### A Tautological Comparison

From the above, we deduce that

- both projective precovers and covers are epic;
- if $M$ admits a projective cover, then it is unique up to isomorphism, and is always a summand of $Q ‚Ü† M$;
- $p : P ‚Ü† M$ is a (pre)cover whenever $p_‚àó : (-, P)_{ùêèùê´ùê®ùê£} ‚Ü† (-|_{ùêèùê´ùê®ùê£}, M)_{\mathcal{A}}$ is.

We are pleased to compare the projective covers with those arising in the functorial context.

{% ex %}
(Idempotent obstacle). There exist cases where an idempotent morphism $e : X ‚Üí X$ does not split. In this case, $\mathrm{im}(-, e)$ is a non-representable finitely generated projective object.
{% endex %}

**Assuming $\mathcal{A}$ is idempotent complete**, we begin with a well-known lemma.

{% pro %}
(Special Yoneda lemma). This is a standard result in representation theory. Let $T'$ be a summand of finite biproducts of $T$. There is an isomorphism

$$\begin{equation}
(T', M)_{\mathcal{A}} ‚Üí ((T,T')_\mathcal{A}, (T, M)_\mathcal{A})_{\mathrm{End}(T)}, \quad f ‚Ü¶ f_‚àó .
\end{equation}$$

The right module structure of $(T,M)$ is given by $f ‚ãÖ Œ± = f ‚àò Œ±$ for $Œ± ‚àà \mathrm{End}(T)$.

{% endpro %}

{% thm %}
(Projectivisation). Let $ùêöùêùùêù (T)$ denote the full subcategory containing $T$ and closed under isomorphisms, finite biproducts and summands. Then there is an equivalence(s):

$$\begin{aligned}
(T,-) : ùêöùêùùêù (T) &‚Üí ùê©ùê´ùê®ùê£ (ùê¶ùê®ùêù _{\mathrm{End}(T)}),\quad T_0 ‚Ü¶ (T, T_0)\\
ùêÄùêùùêù (T) &‚Üí ùêèùê´ùê®ùê£ (ùêåùê®ùêù _{\mathrm{End}(T)}),\quad T_0 ‚Ü¶ (T, T_0).
\end{aligned}$$

{% pf %}
The first equivalence holds due to special Yoneda lemma. The full subcategory $ùê©ùê´ùê®ùê£ ‚Ü™ ùêèùê´ùê®ùê£$ is a $\{‚àê, ùêíùê¶ùêù\}$-closure of the $ùê©ùê´ùê®ùê£ ‚Ü™ (ùêèùêíùê°(ùê©ùê´ùê®ùê£ ^{\mathrm{op}}))^{\mathrm{op}}$. Note that direct summands are determined by image of idempotent morphism in endomorphisms algebras. Hence the second isomorphism holds.
{% endpf %}

{% endthm %}

The tautological comparison is as follows.

{% thm %}
We assume idempotent completeness. Let $p : P ‚Üí M$ be a $\mathcal{P}$-precover, then the following are equivalent.

1. $p : T ‚Üí M$ admits a $\mathcal{P}$-cover;
2. $p_‚àó : (-, P)_{\mathcal{P}} ‚Ü† (-|_{\mathcal{P}}, M)$ admits a projective cover;
3. $p_‚àó : (-, P)_{ùêöùêùùêù (P)} ‚Ü† (-|_{ùêöùêùùêù (P)}, M)$ admits a projective cover;
4. $p_‚àó : (P, P)_{\mathrm{End}(P)} ‚Ü† (P, M)_{\mathrm{End}(P)}$ admits a projective cover.

Note that the last three are projective precovers as they are epimorphisms.

{% pf %}
(1 ‚Üî 2) follows from the definition. In the case of $1$, a $\mathcal{P}$-cover must be a summand of $p$. Since there is no differences between $\mathcal{P}$-cover and $ùêöùêùùêù (P)$-cover for $M$, we show (1 ‚Üî 3). (3 ‚Üí 4) is given by the special Yoneda lemma. It remains to show (4 ‚Üí 1).
\\
When $p_‚àó : (P, P)_{\mathrm{End}(P)} ‚Ü† (P, M)_{\mathrm{End}(P)}$ admits a projective cover, it takes the form

$$\begin{equation}
(P, P)_{\mathrm{End}(P)} ‚Ü† Q ‚Ü† (P, M)_{\mathrm{End}(P)}.
\end{equation}$$

Note that $Q ‚âÉ (P, P_0)_{\mathrm{End}(P)}$ for $P_0 ‚àà ùêíùê¶ùêù (P)$ by projectivisation. With the special Yoneda lemma, we may write the mapping sequence as

$$\begin{equation}
(P, P)_{\mathrm{End}(P)} \overset {a_‚àó } ‚Ü† (P, P_0)_{\mathrm{End}(P)} \overset {b_‚àó } ‚Ü† (P, M)_{\mathrm{End}(P)}.
\end{equation}$$

We show that $b : P_0 ‚Üí M$ is a $\mathcal{P}$-cover. By special Yoneda lemma, $(ba)_‚àó = p _‚àó$ yields $ba = p$. As $a$ is split epic, $b$ and $p$ factors through each other, **which shows that $b$ is a $\mathcal{P}$-precover**. If there exists $b ‚àò Œ≥ = b$, we have $b_‚àó ‚àò Œ≥ _‚àó = b_‚àó$. Now $Œ≥ _‚àó$ is an isomorphism as $b_‚àó$ is a projective cover. By projectivisation, $Œ≥$ is an automorphism. Hence $b$ is a $\mathcal{P}$-cover.

{% endpf %}

{% endthm %}

It is a local property to exhibit a projective cover. For idempotent complete cases, a $\mathcal{P}$-precover $P ‚Üí M$ admits a $\mathcal{P}$-cover when $\mathrm{End}(P)$ is right semi-perfect (where f.g. modules admits a projective cover).

### Example: vNR Rings

A ring $R$ with such that $ùêèùê´ùê®ùê£ = ùêåùê®ùêù_R$ is known as a semisimple ring. In comparison with this, the ring $R$ with $ùêÖùê•ùêöùê≠ = ùêåùê®ùêù _R$ is called vNR (von Neumann regular).

- The property of being either semisimple or vNR is a **symmetric property**, i.e., there is no need to distinguish left and right modules.
- This is not the original definition in sense of von Neumann; but it is a beautiful coincidence that these two definitions coincide.

{% thm %}
The followings are equivalent definition for $R$ to be a vNR ring. The first definition shows being vNR is a **symmetric property**. We proof for right modules only.

### Example: Torsion Theory

***The covering need not be epimorphisms!*** A trivial example is that every additive category admits a $\{0\}$-covering. A general kind of examples is known as the *torsion theory*.

Unwinding a brief introduction, we enumerate some closure properties of a torsion class as follows.

{% def %}
(Torsion theory). Let $\mathcal{A}$ be an Abelian category which is **well-powered** and **subobjects are closed under transfinite filtrations**. Then the following are equivalent for defining a torsion theory $(\mathcal{T}, \mathcal{F})$:

1. There is a subfunctor $t ‚äÜ 1_\mathcal{A}$ such that $t ‚àò t = t$ and $t(\frac{M}{tM}) = 0$. Here $t ‚àò t = t$ means that $t(tM) = tM$ when identifying subobjects of $tM$ as that of $M$. Set $\mathcal{T} = \operatorname{im} t$ and $\mathcal{F} = \ker t$.

3. $\mathcal{T}^o = \mathcal{F}$ and $\mathcal{T} = {}^o \mathcal{F}$, or other equivalent form given by Galois connection.
4. $\mathcal{F} = \mathcal{T}^o$, and $\mathcal{T}$ is closed under taking transfinite filtrations (including extensions) and quotient objects.
5. $\mathcal{F} = \mathcal{T}^o$, and there is a class of objects $\mathcal{X}$ such that $\mathcal{T}$ consists of objects which are transfinite filtrations of quotient objects in $\mathcal{X}$.

{% pf %}
We explain $1 \leftrightarrow 2 \rightarrow 3 \rightarrow 4 \leftrightarrow 5$ in a few words, and show $5 \rightarrow 1$ (which requires supplementary conditions).\\
($1 \leftrightarrow 2$). A canonical injection refers to a canonical short exact sequence $t$. ($2 \rightarrow 3$). For instance, we show $\mathcal{F} = \mathcal{T} ^o$. Conversely, any particular class $ùêÖùê¢ùê• (\text{quotients of } \ \mathcal{X})$ is closed under transfinite filtrations and quotient objects.
\\
($4 \rightarrow 2$). We show $X ‚àà \mathcal{A}$ has a maximal subobject in $\mathcal{T}$, which is also the greatest. Let $\mathcal{S}$ denote the set of subobjects of $X$ lying in $\mathcal{T}$. $0 ‚àà \mathcal{S}$ is non-empty. Notice that,

- Every chain in $\mathcal{S}$ has an upper bound, as $\mathcal{T}$ is closed under transfinite filtrations.
- Any two maximal objects $T_1, T_2 ‚àà \mathcal{S}$ coincide. If not, then $(T_1 + T_2) ‚àà \mathcal{T}$, a contradiction.

Let $tX$ denote such a maximal subobject in $\mathcal{T}$; we complete the proof.

{% endpf %}

{% enddef %}

We call $\mathcal{T}$ a torsion class, and $\mathcal{F}$ a torsion-free class.

The above are equivalent for either $ùêåùê®ùêù _R$ or $C(R)$. Note that these categories are well-powered and cocomplete.

## Set-theoretic Consideration

We show few results of the existence of (special) cover/envelope, provided the existence of (special) precover/preenvelope. The analysis holds for general well-powered Abelian category closed under some (co)limits. To conclude, we have the following theorem.

{% thm %}
Suppose $(\mathcal{C},\mathcal{F})$ is a complete cotorsion theory such that either

- $\mathcal{C}$ is closed under filtered colimits, or
- $\mathcal{F}$ is closed under cofiltered limits.

Then $\mathcal{C}$ is a covering class and $\mathcal{F}$ is an enveloping class.
{% endthm %}

### Precover ‚áí Cover

Let $\mathcal{P}$ is a replete class of modules or chain complexes. We assume **$\mathcal{P}$ is closed under filtered colimits** and **summands**, e.g. $\mathcal{P} = ùêÖùê•ùêöùê≠$ or the entire category.

In this case, it is feasible to weaken the definition of $\mathcal{P}$-cover.

{% lem %}
Let $p : P ‚Üí M$ be a $\mathcal{P}$-precover. It is a $\mathcal{P}$-cover, iff the following equivalent conditions holds:

1. for any other morphism $q$ from $\mathcal{P}$ to $M$, $q ‚àò Œ± = p$ implies $Œ±$ is monic,
2. for any other $\mathcal{P}$ precover $p$, $p' ‚àò Œ± = p$ implies $Œ±$ is monic,
3. $p ‚àò Œ± = p$ implies $Œ±$ is monic,
4. $p ‚àò Œ± = p$ implies $Œ±$ is an isomorphism.

{% pf %}
(1 ‚Üí 2 ‚Üí 3) is clear. For (3 ‚Üí 1), we take any such $q$ with $q ‚àò Œ± = p$. Since $p$ is a $\mathcal{P}$-precover, there is and $q = p ‚àò Œ≤$. We see $(Œ≤ ‚àò Œ±)$ is monic. Hence $Œ±$ is monic. (4 ‚Üí 2) is clear.\\
We show (3 ‚Üí 4). If not, then there exists a stricly injection $i : P ‚Üí P$ such that $p ‚àò i = p$. We shall construct a transfinite composition which terminates at $Œ∫ ‚â´ |P|$:

$$\begin{equation}
P_0 \xrightarrow {f_{1,0}} P_1  \xrightarrow {f_{2,1}} P_2 ‚Üí \cdots P_Œ≥ \xrightarrow {f_{Œ≥ +1,Œ≥ }} \cdots P_{Œ∫}.
\end{equation}$$

For any $\mathcal{P}$-precover $p_Œ≥ : P_Œ≥ ‚Üí M$, we define $P_{Œ≥ + 1} := P$ and $p_{Œ≥ + 1} = p$. In particular,

- when $P_Œ≥ = P$, we take $f_{Œ≥ + 1, Œ≥ } = i$,
- when $Œ≥$ is a limit ordinal, then $p_{Œ≥}$ is induced by universal property of colimit. By closure property of $\mathcal{P}$, we see $P_Œ≥ ‚àà \mathcal{P}$.

Clearly, $p_{Œ≥} = p$ when $Œ≥$ is not a limit ordinal. We take the subsystem $S := \{P_Œ≥ ‚à£ p_Œ≥ = p\}$ consisting of strict monomorphisms. As a result, the subobjects of $P$ has cardinality $Œ∫$. Since $Œ∫$ is arbitrary large,  a contradiction.

{% endpf %}

### Envelope ‚áí Preenvelope

Suppose $\mathcal{E}$ is a class of modules closed under summand, along with the following concession.

{% def %}
(Concession). If there is a preenvelope $i : M ‚Üí E$, then any morphism from $M$ to the cofiltered limit $\varprojlim^{cof} E_‚àô$ factors through $i$.

- In particular, the concession holds if $\mathcal{E}$ is closed under cofiltered limits.

{% enddef %}

We denote $\widetilde {\mathcal{E}}$ as the $\varprojlim ^{cof}$-completion of $\mathcal{E}$, and introduce the *relative cokernel* (or *relative image* for convenience). 

{% def %}
(Image). For a general morphism of the type $j : M ‚Üí \widetilde {\mathcal{E}}$, we define

$$\begin{equation}
\mathrm{IM}(j) := \{\operatorname{im}(Œ± ) ‚à£ Œ± : i ‚Üí j,\ i \ \text{ is a preenvelope}\}.
\end{equation}$$

We also have the *cobase change*. Say $Œ± ‚àà \mathrm{IM}(j)$ is a **relative envelope**, provided

$$\begin{equation}
Œ± _‚ôØ : \mathrm{IM}(i) ‚Üí \mathrm{IM}(j), \quad \operatorname{im}(?) ‚Ü¶ \operatorname{im}(Œ± ‚àò ?),\ \text{such that} \ \operatorname{im}(Œ± _‚ôØ ) = \{\operatorname{im}(Œ± )\}.
\end{equation}$$

{% enddef %}

We show the existence of relative envelope.

{% lem %}
Any morphism of the type $j : M ‚Üí F ‚àà \widetilde {\mathcal{E}}$ has a relative envelope.

{% pf %}
If not, then there is a transfinite composition of towers

$$\begin{equation}
\cdots \overset {f_{Œ≥ +1, Œ≥+0}}‚Üí  (E _{Œ≥ +1}, i _{Œ≥ +1}) \overset {f_{Œ≥,Œ≥+1}} ‚Üí (E_Œ≥, i _Œ≥ ) \cdots \overset {f_{1,2}}‚Üí  (E_1, i _1) \overset {f_{0,1}}‚Üí (E_0, i _0) = (F, j),
\end{equation}$$

such that

1. for any $Œ≤$, $i_Œ≤ ‚àò Œ± _Œ≤ = i_{Œ≤ +1}$;
2. if $Œ≤$ is a successor ordinal, then $i_Œ≤$ is a preenvelope;
3. if $Œ≤$ is a limit ordinal, then $i_Œ≤$ is induced by limit;
4. $\{\operatorname{im}(f_{0, Œ≥})\}$ strictly descends when composing $f_{Œ≤ +1, Œ≤ +2}$.

Let $Œ∫ ‚â´ |F|$, we see the contradiction.

{% endpf %}

{% endlem %}
